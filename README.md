This Is a news NLP project which can classify, translate and summarize news articles with many more features along with a small database. 
You can get the ui file and the classifer contents from this repo but the project is incomplete without the translator and summarizer model which you can find at the 
following hugging face link https://huggingface.co/garvitupdy/News-NLP-Project/tree/main.
You can download the models locally and then made the following folder structure for the project to work -
Extract them in your project folder in the following structure
   your_project/
   ├── ui.py
   ├── summarizer/  (copy the downloaded summarizer model files here)
   └── translator/  (copy the downloaded translator model files here)
   |__classifier/ (copy the downloaded classifer model files here)
